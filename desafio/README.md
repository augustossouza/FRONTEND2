

Conteúdo:
- index.html — página pessoal
- contato.html — página de contato com formulário
- extra.html — página extra (relato)
- css/styles.css — estilos
- assets/ — imagens e outros assets
- form.html -  campos com nome, sobrenome, email e idade.
- confirmation.html - campo de confirmação do form.html


1. Baixe/clique em `index.html` no seu navegador (duplo clique) ou abra com `File > Open` no navegador.
2. Todas as páginas usam caminhos relativos; mantenha a estrutura de pastas.
